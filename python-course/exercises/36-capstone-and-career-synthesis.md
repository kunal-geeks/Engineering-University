## Capstone deliverables checklist

- [ ] Working system deployed locally (Docker Compose minimum)
- [ ] Architecture diagram (components + data flow)
- [ ] 2+ ADRs documenting key decisions
- [ ] Runbook: deploy, rollback, on-call common issues
- [ ] SLO defined with one dashboard or metric screenshot
- [ ] Test suite: unit + at least one integration test
- [ ] 15-minute walkthrough recording or slide deck
