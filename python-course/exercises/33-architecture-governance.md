# Exercise Solutions — Lesson 33

## RFC vs PR review matrix

| Change | Process |
|--------|---------|
| Bug fix, test addition | PR |
| New endpoint, backward compatible | PR + design note |
| New service | RFC required |
| Breaking API / schema | RFC + migration plan |
| Org-wide standard change | RFC + leadership sign-off |

See lesson for ADR and deprecation plan templates.
